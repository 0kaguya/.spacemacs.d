# .spacemacs.d

config of spacemacs. place it at `~/.spacemacs.d`.

uncomment lines in `spacemacs/user-init` to set the [tuna][tuna] mirror.

## Requirement

### path executable

- haskell
  - `apply-refact`
  - `hasktags`
  - `hlint`
  - `hoogle`
  - `stylish-haskell`
  
- python
  - `autoflake`
  - `flake8`
  - `isort`
  - `jedi`
  - `json-rpc`
  - `service_factory`
  - `yapf` 

[tuna]: https://mirrors.tuna.tsinghua.edu.cn
