# .spacemacs.d

config of spacemacs. place at `~/.spacemacs.d`.

for upstream package repository uses the [tuna][tuna] mirror.

## Requirement

### path executable

- haskell
  - `apply-refact`
  - `hlint`
  - `stylish-haskell`
  - `hasktags`
  - `hoogle`

[tuna]: https://mirrors.tuna.tsinghua.edu.cn
